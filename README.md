# AIGC H5 作品集提交包

## 作品集结构

- `index.html`：作品集首页，标题为姓名，包含 1 个作品的图片和网页链接。
- `work/index.html`：主作品《空白画布急诊室》H5 可交互页面。
- `assets/work-preview.svg`：作品图片，用于作品集首页展示。

## GitHub Pages 部署方法

1. 新建 GitHub 仓库，推荐仓库名：`canva-ai-portfolio`。
2. 把本文件夹里的所有内容上传到仓库根目录。
3. 进入仓库：`Settings → Pages`。
4. Source 选择：`Deploy from a branch`。
5. Branch 选择：`main`，Folder 选择：`/root`。
6. 保存并等待 1–2 分钟。

## 提交链接说明

假设你的 GitHub 用户名是 `username`，仓库名是 `canva-ai-portfolio`：

- 作品集链接：`https://username.github.io/canva-ai-portfolio/`
- 作品链接：`https://username.github.io/canva-ai-portfolio/work/`

## 修改姓名

打开 `index.html`，搜索 `王紫玫`，替换为你的真实提交姓名即可。

## 作品说明

《空白画布急诊室》以 Canva 可画为品牌对象，将用户面对设计任务时的“空白画布、排版混乱、配色失控、时间告急”等问题转化为 AI 设计急诊室的互动叙事。页面包含设计症状诊断、AI 急诊舱、任务选择、可编辑生成器、Before/After 对比、导出 PNG 与跳转 Canva 可画等交互功能。
